# Audio Assets

Place royalty-free audio files here:

## Music Loops (Background Music)
- `menu.mp3` — Gentle menu/home screen music
- `fantasy.mp3` — Fantasy kingdom genre loop
- `space.mp3` — Outer space genre loop
- `underwater.mp3` — Underwater world genre loop
- `jungle.mp3` — Jungle safari genre loop

## Sound Effects
Sound effects are loaded from external CDN (Mixkit) in `hooks/useSoundEffects.ts`.

## Sources for Royalty-Free Audio
- https://pixabay.com/music/ (download and save locally)
- https://mixkit.co/free-stock-music/children/
- https://www.chosic.com/free-music/children/

All files should be:
- MP3 format
- Under 2MB each
- 30-120 seconds duration
- Suitable for looping
