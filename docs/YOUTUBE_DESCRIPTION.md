🏗️ TECH STACK
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AWS Services:
• Amazon DynamoDB — User data, progress, characters, quests (PAY_PER_REQUEST)
• Amazon Bedrock (Nova Pro) — Vision analysis + story generation
• Amazon Bedrock (Nova Canvas) — Character & scene image generation
• Amazon Bedrock (Nova Lite) — Content moderation for child safety
• Amazon Polly Neural — Text-to-speech narration (Voice: Ruth)
• Amazon S3 — Asset storage (images, audio, drawings)
• Amazon EC2 — Backend hosting

Frontend & Deployment:
• Vercel — Frontend CDN deployment
• Next.js 14 + React 18 + TypeScript + Tailwind CSS

Backend:
• FastAPI (Python) — AI orchestration & data persistence

🎯 WHAT IT DOES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Child draws on a digital canvas (magic brushes, stickers, stamps)
2. Amazon Nova Pro analyzes the drawing and describes the character
3. Amazon Nova Canvas generates an illustrated character image
4. Child picks a life lesson (sharing, kindness, courage, etc.)
5. Amazon Nova Pro generates an 8-scene interactive story quest
6. Amazon Nova Canvas illustrates each scene
7. Amazon Polly narrates every scene aloud with expressive voice
8. Amazon DynamoDB persists all progress, characters, and achievements
9. Gamification: XP, levels, coins, streaks, and achievements
